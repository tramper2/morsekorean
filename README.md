# 📟 한글 모르스 부호 변환기 (MorseKorean)

한글, 영문, 숫자를 모르스 부호로 변환하고 오디오 신호로 재생하는 웹 기반 도구입니다.

## ✨ 주요 기능

### 1. 실시간 양방향 변환
- **텍스트 → 모르스**: 한글, 영문, 숫자를 입력하면 즉시 모르스 부호로 변환
- **모르스 → 텍스트**: 모르스 부호를 직접 입력하면 한글로 자동 조합
- Hangul.js 라이브러리를 사용하여 한글을 자모(자음, 모음)로 정확하게 분리

### 2. SKATS 표준 준수
국제 통신 표준인 **한글 전송 부호(SKATS)** 기준을 적용하여 정확한 변환을 지원합니다.

**지원 문자:**
- **자음**: ㄱ, ㄴ, ㄷ, ㄹ, ㅁ, ㅂ, ㅅ, ㅇ, ㅈ, ㅊ, ㅋ, ㅌ, ㅍ
- **모음**: ㅏ, ㅑ, ㅓ, ㅕ, ㅗ, ㅛ, ㅜ, ㅠ, ㅡ, ㅣ, ㅐ, ㅔ
- **숫자**: 0-9
- **기호**: ?, ., ,, /

### 3. 오디오 신호 재생
- 웹 오디오 API를 활용하여 실제 무선통신 장비의 신호음을 재현
- **주파수**: 700Hz 사인파
- **타이밍**:
  - 점(·): 0.1초
  - 선(−): 0.3초 (점의 3배)
  - 부호 간 간격: 0.1초
  - 문자 간 간격: 0.3초

## 🚀 사용 방법

### 1. 텍스트를 모르스 부호로 변환
1. 상단의 "텍스트 (한글/영문/숫자)" 입력창에 원하는 텍스트 입력
2. 하단의 "모르스 부호" 창에서 자동으로 변환된 모르스 부호 확인
3. "🔊 신호 듣기" 버튼을 클릭하여 오디오 신호 재생

### 2. 모르스 부호를 텍스트로 변환
1. 하단의 "모르스 부호" 입력창에 직접 모르스 부호 입력
2. **형식**: 부호와 부호는 **1칸 띄우고**, 글자 사이는 **3칸** (또는 **/** 사용)
   - 예: `.-.-  ...-  ..-. ` 또는 `.-.-/...-/..-. `
3. 상단의 "텍스트" 창에서 자동으로 변환된 텍스트 확인
